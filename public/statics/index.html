Note is a practice tool designed for you to keep track of what songs you are learning, and what tempo you are at. It also has a built in music player, and stores the songs themselves. This allows us as musicians to keep track of how well we are doing with learning our songs.

Getting Started

Before you can start using the music feature, you will need an account. Once you have made an account, you can start uploading songs to work with. Note will store the songs for you so you don't have to constantly reload the songs.
