Tutorial


Below is a basic tutorial on what each of the six table entries means and how to properly enter the data to make a Magic: the Gathering Card. To navigate to a specific trait, click on a link below:

  • Name
  • Mana Cost
  • Card Type
  • Abilities
  • Flavor Text
  • Rarity

Name

The name of the card is simply the name given to it. Like usernames, cards have unique names. The name can be found at the top left of the card.


Mana Cost

This indicates how much it costs to use the card. The number represents the number of any kind of mana to pay for it, while each of the color symbols represents how much of a specific color mana is required to pay for it. In the image to the right, it has a mana cost of 1 and a red, so it costs 2 mana to play it. The mana cost is found on the top right of the card, next to the name.


Card Type

This indicates the type of card it is. In MTG, the following card types exist:

  • Creature
  • Instant
  • Sorcery
  • Land
  • Planeswalker
  • Artifact
  • Enchantment

In addition, there are some subtypes, and cards can have multiple types (for example, many cards are both an artifact and a creature). The card type is found right below the card's art, around the middle left of it.


Abilities

This indicates what the card does exactly. The abilities can include a wide range of information. The abilities are found in the center of the card, right below the card type.


Flavor Text

This is an optional part of a card. Flavor text does not add anything to the game mechanically, but it adds to the story and is recommended when crafting cards that tell a story.


Rarity

The rarity is used to indicate how common the card is when opening it up in a pack of random cards. It does not change how the card works, but it is needed when making a set of cards. The more common the card is, the simpler it should be. Below are the four rarities in the game:

  • Common (indicated with black text)
  • Uncommon (indicated with silver text)
  • Rare (indicated with gold text)
  • Mythic (indicated with red text)

Rarity is found to the right of the card type.
